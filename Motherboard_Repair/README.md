# Sharp MZ-80B Motherboard Repair
My 80B arrived non-functional, no nuthin' working.<br>

It appears the motherboard had suffered some water damage as there was a lot of rusty ICs in sockets.  So these needed replacing.<br>

The socketed ICs that I need to replace:

| Part   | Original    | Type       | Replacement    |
|--------|-------------|------------|----------------|
| IC2    | G#1 14297   | Gate Array | No replacement |
| IC3    | LH0080A     | CPU        | Z80            |
| IC8    | IPL MB 8516 | Boot ROM   | 2716           |
| IC19   | D8253C      | PIT        | 8253           |
| IC23   | D8255C      | PPI        | 8255           |
| IC24   | LH0081A     | PIO        | Z80-PIO        |
| IC41   | TMM2016P-1  | 2KB SRAM   | 6116           |
| IC42   | G#2 14298   | Gate Array | No replacement |
| IC43   | G#3 14299   | Gate Array | No replacement |
| IC44   | TMM323D-1   | CG ROM     | 2716           |
| RAM(I) | LH4116-3    | 16kb DRAM  | Custom board   |
| RAM(II)| LH4116-3    | 16kb DRAM  | Custom board   |
